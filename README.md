# Housing-in-Buenos-Aires
Apartment Price Prediction — Buenos Aires  This project focuses on building a machine learning model to predict apartment prices in Buenos Aires, Argentina. It covers the full data science workflow, from data preprocessing to model deployment and interactive visualization.

What this project demonstrates:
Building a Linear Regression model for real estate price prediction
Creating a data preprocessing pipeline:
Handling missing values
Encoding categorical variables
Improving model performance and reducing overfitting
Developing an interactive dashboard to explore predictions dynamically
Key Features :
Clean and structured data pipeline
Model training and evaluation using real-world data
Interactive UI to test predictions based on user inputs (area, location, neighborhood)
Visualization of predictions on a map
Tech Stack :
Python
pandas & NumPy
scikit-learn
Plotly
ipywidgets
Goal :

To provide an end-to-end example of how machine learning can be applied to real estate data, while building a reusable and interactive prediction tool.
